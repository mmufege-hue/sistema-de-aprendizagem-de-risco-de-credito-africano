# sistema-de-aprendizagem-de-risco-de-credito-africano
sistema de aprendizagem de risco de credito africano é uma plataforma completa e interativa para desenvolvimento, explicação e simulação de modelos de risco de crédito, construída inteiramente em Python e Streamlit, com foco especial em mercados emergentes como África e ambientes com baixa disponibilidade de dados.
