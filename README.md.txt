# 💳 CreditMasterLab
### End-to-End Credit Risk & Scoring System for Emerging Markets  
**Built with Streamlit • Python • Machine Learning • IFRS 9 • Explainability & Decisioning**

![Banner](assets/banner.png)

CreditMasterLab is a full end‑to‑end credit risk platform designed for **learning, simulation, model development, credit policy design and stress testing**, with a special focus on **African & frontier markets** where traditional data is scarce.

This project integrates:

✅ Machine learning credit scoring  
✅ WOE/IV, ChiMerge binning, Scorecards  
✅ AfricaMode – scoring for clients without banking history  
✅ IFRS 9 stress testing (macro scenarios)  
✅ Explainability Engine (SHAP + Policy Rules)  
✅ Decision Engine (Approve/Reject/Review)  
✅ What‑If Simulator  
✅ Scorecard Exporter (Excel & PDF)  
✅ Full Pipeline Automation  
✅ Deploy-ready Streamlit application  

---

# 🚀 Features

### ✅ 1. **Data Engineering for Credit Risk**
- Automatic dataset generator  
- Cleaning, EDA and statistical summaries  
- Alternative data generation (mobile, georisk, proxy income)

### ✅ 2. **Credit Modeling**
- Logistic Regression for PD estimation  
- WOE/IV calculation  
- ChiMerge supervised binning  
- Model performance: AUC, KS, ROC, metrics dashboard  

### ✅ 3. **WOE Pipeline**
- Automatic mapping  
- WOE transformation  
- Full WOE-based modeling pipeline  

### ✅ 4. **Scorecard Generator**
- Generate scorecards from logistic regression  
- Points per variable/bin  
- Export to **Excel** or **PDF**

### ✅ 5. **IFRS 9 Stress Testing**
- Macro scenarios (Inflation, GDP, FX)  
- PD shifts, LGD, EAD  
- ECL calculation  
- Scenario comparison charts  

### ✅ 6. **Africa Mode – Scoring Without Banking History**
- Mobile behavior features  
- Proxy income  
- Top-up patterns  
- Geo-risk maps  
- AfricaScore (0–100)

### ✅ 7. **Decision Engine**
- Automatic credit decisions  
- Approve / Review / Reject  
- Limit assignment  
- Fully configurable credit policy studio  

### ✅ 8. **Explainability Engine**
- SHAP values for model explanation  
- Rule-based explanation for policy decisions  
- Combined interpretable reasoning  

### ✅ 9. **What‑If Simulator**
Simulate:
- “E se o AfricaScore melhorar?”
- “E se a renda subir?”
- “E se mudar de zona geográfica?”
- Updated PD, decision, and credit limit  

### ✅ 10. **Pipeline End-to-End (One Button)**
Dataset → WOE → Model → PD → AUC/KS → Stress Test → Scorecard → PDF
